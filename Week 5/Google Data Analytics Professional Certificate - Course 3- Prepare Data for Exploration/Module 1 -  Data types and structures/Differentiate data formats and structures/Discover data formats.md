# Data Types and Data Formats

## Main Idea

As a Data Analyst, you need to understand different types of data and data formats so you can choose the right data for your analysis.

Different questions require different kinds of data.

---

# Why Data Types Matter

Data can be compared, categorized, counted, measured, stored, and organized in different ways.

Understanding data types helps analysts know:

- what kind of analysis is possible
- how to organize the data
- how to compare data
- what tools to use
- whether the data fits the business question

---

# Movie Data Example

The lesson uses a spreadsheet of movie data to explain different types of data.

Examples of movie data include:

- movie titles
- cast members
- movie genres
- budgets
- box office revenue
- run time

Different columns in the spreadsheet represent different types of data.

---

# Qualitative Data

## Definition

**Qualitative data** is data that cannot be counted, measured, or easily expressed using numbers.

It usually describes qualities, names, categories, or labels.

## Examples from the Movie Spreadsheet

- movie titles
- cast members
- movie genres
- descriptions

## Simple Explanation

Qualitative data tells you what something is like, not how much of it there is.

---

# Quantitative Data

## Definition

**Quantitative data** is data that can be measured or counted and expressed as a number.

It represents a quantity, amount, or range.

## Examples from the Movie Spreadsheet

- movie budget
- box office revenue
- number of tickets sold
- movie runtime

## Simple Explanation

Quantitative data tells you how much, how many, or how long.

---

# Discrete Data

## Definition

**Discrete data** is quantitative data that is counted and has a limited number of values.

It usually cannot be divided into smaller meaningful parts between values.

## Example

Movie budget and box office revenue are examples of discrete data when expressed in dollars and cents.

For example, money can be counted using dollars and cents, and there cannot be a value between one cent and two cents.

## Simple Explanation

Discrete data is counted.

---

# Continuous Data

## Definition

**Continuous data** is quantitative data that can be measured and can have many possible decimal values.

It can often be measured using tools such as timers, scales, or sensors.

## Example

A movie runtime could be written as:

- 110.0356 minutes

This value can include many decimal places.

## Simple Explanation

Continuous data is measured and can have very detailed decimal values.

---

# Nominal Data

## Definition

**Nominal data** is a type of qualitative data that is categorized without a set order.

It does not have a natural sequence or ranking.

## Example

If people are asked whether they have watched a movie, their answers might be:

- Yes
- No
- Not sure

These categories do not have a specific order.

## Simple Explanation

Nominal data names or labels categories without ranking them.

---

# Ordinal Data

## Definition

**Ordinal data** is a type of qualitative data that has a set order or scale.

## Example

If people rate a movie from 1 to 5, the ratings have an order.

A rating of 5 is higher than a rating of 4, and a rating of 4 is higher than a rating of 3.

## Simple Explanation

Ordinal data shows rank or order.

---

# Internal Data

## Definition

**Internal data** is data that lives within a company’s own systems.

It is generated, collected, or stored by the organization itself.

## Example

If a movie studio collected data only about its own movies using its own systems, that would be internal data.

## Benefits

Internal data is often:

- easier to collect
- more reliable
- easier to verify
- already connected to the company’s business needs

---

# External Data

## Definition

**External data** is data that lives outside an organization and is generated by outside sources.

## Example

If a movie studio uses data from other studios or outside databases about movies it did not produce, that is external data.

## Why External Data Is Useful

External data is useful when analysis depends on many sources or when the organization does not have enough internal data.

---

# Structured Data

## Definition

**Structured data** is data that is organized in a clear format, usually rows and columns.

## Examples

Structured data can be stored in:

- spreadsheets
- tables
- relational databases

## Why Structured Data Is Useful

Structured data is usually easier to:

- search
- sort
- filter
- analyze
- organize
- compare

## Simple Explanation

Structured data has a clear framework.

---

# Unstructured Data

## Definition

**Unstructured data** is data that is not organized in an easily identifiable format.

It does not fit neatly into rows and columns.

## Examples

- audio files
- video files
- images
- social media posts
- emails
- open-ended text responses

## Important Note

Unstructured data may still have internal structure, but it is not organized in a simple table format.

## Simple Explanation

Unstructured data does not fit neatly into rows and columns.

---

# Structured Data vs Unstructured Data

| Data Type | Meaning | Examples |
|---|---|---|
| Structured Data | Organized in rows and columns | Spreadsheets, tables, relational databases |
| Unstructured Data | Not organized in a clear table format | Audio, video, images, free text |

---

# Internal Data vs External Data

| Data Type | Meaning | Example |
|---|---|---|
| Internal Data | Data from inside the organization’s own systems | A movie studio’s data about its own films |
| External Data | Data from outside the organization | Movie data from other studios or external databases |

---

# Qualitative vs Quantitative Data

| Data Type | Meaning | Examples |
|---|---|---|
| Qualitative Data | Descriptive data that cannot easily be counted or measured | Movie titles, cast members, genres |
| Quantitative Data | Numeric data that can be counted or measured | Budget, revenue, runtime |

---

# Discrete vs Continuous Data

| Data Type | Meaning | Example |
|---|---|---|
| Discrete Data | Counted data with limited values | Money in dollars and cents |
| Continuous Data | Measured data that can include many decimal values | Movie runtime in minutes |

---

# Nominal vs Ordinal Data

| Data Type | Meaning | Example |
|---|---|---|
| Nominal Data | Categories with no set order | Yes, No, Not sure |
| Ordinal Data | Categories with a ranked order | Movie rating from 1 to 5 |

---

# Big Lesson

Data comes in many types and formats.

To analyze data correctly, you need to understand what kind of data you are working with and how it is organized.

---

# Key Takeaways

- Qualitative data describes names, categories, or qualities.
- Quantitative data can be counted or measured as numbers.
- Discrete data is counted and has limited values.
- Continuous data is measured and can include many decimal places.
- Nominal data is categorized without a set order.
- Ordinal data is categorized with a set order or scale.
- Internal data comes from within a company’s own systems.
- External data comes from outside the organization.
- Structured data is organized in rows and columns.
- Unstructured data does not fit neatly into rows and columns.
- Spreadsheets and relational databases store structured data.
- Audio and video files are examples of unstructured data.
- Understanding data formats helps analysts choose the right data for the question.

---

# Extra Important Notes

- Movie titles are qualitative data because they are names.
- Movie budgets and box office revenue are quantitative data because they are numbers.
- Ratings from 1 to 5 are ordinal because they have order.
- Yes, No, and Not sure are nominal because they are categories without order.
- Structured data is usually easier to analyze than unstructured data.
- Data type affects what kind of analysis you can perform.
- The right data format makes analysis easier and more reliable.

---

# Final Summary

This lesson explains different types and formats of data using a movie spreadsheet example. Qualitative data includes descriptive values such as movie titles and cast members, while quantitative data includes measurable numbers such as budgets and box office revenue. Quantitative data can be discrete, like money, or continuous, like runtime. Qualitative data can be nominal, with no set order, or ordinal, with a ranked order. Data can also be internal or external depending on where it comes from. Finally, data can be structured, like spreadsheets and databases, or unstructured, like audio and video files. The main lesson is that understanding data types and formats helps Data Analysts choose and use the right data for their analysis.
