# Data Format Examples

## Main Idea

A **data format** is the way data is organized, stored, or presented.

Choosing the right data format helps Data Analysts manage, understand, and use data effectively.

Just like an advertisement can appear as a print ad, billboard, or commercial, data can also appear in different formats depending on how it needs to be used.

---

# Why Data Format Matters

The format of a dataset affects how easily you can:

- organize the data
- analyze the data
- compare values
- identify patterns
- store the data
- share the data
- use the data to answer business questions

### Important Lesson

Choosing the right data format helps you use data in the best way possible.

---

# 1. Primary Data vs Secondary Data

## Primary Data

**Primary data** is data collected by a researcher directly from first-hand sources.

This means the person or organization using the data collected it themselves.

### Examples

- data from an interview you conducted
- data from a survey returned by 20 participants
- data from questionnaires completed by a group of workers

### Simple Explanation

Primary data is data you collect yourself.

---

## Secondary Data

**Secondary data** is data gathered by other people or from other research.

This means the data already existed before you used it.

### Examples

- data bought from a local data analytics firm’s customer profiles
- demographic data collected by a university
- census data gathered by the federal government

### Simple Explanation

Secondary data is data someone else collected.

---

# Primary vs Secondary Data

| Data Type | Definition | Examples |
|---|---|---|
| **Primary Data** | Collected directly from first-hand sources | Interviews, surveys, questionnaires |
| **Secondary Data** | Gathered by other people or from other research | Census data, university research, purchased customer profiles |

---

# 2. Internal Data vs External Data

## Internal Data

**Internal data** is data stored inside a company’s own systems.

This data is generated by the organization itself.

### Examples

- employee wages tracked by HR
- sales data by store location
- product inventory levels across distribution centers

### Simple Explanation

Internal data comes from inside the organization.

---

## External Data

**External data** is data stored outside a company or organization.

This data comes from outside sources.

### Examples

- national average wages for positions in your organization
- credit reports for customers of an auto dealership

### Simple Explanation

External data comes from outside the organization.

---

# Internal vs External Data

| Data Type | Definition | Examples |
|---|---|---|
| **Internal Data** | Data stored inside a company’s systems | HR wages, sales data, inventory levels |
| **External Data** | Data stored outside a company | National wage data, credit reports |

---

# 3. Continuous Data vs Discrete Data

## Continuous Data

**Continuous data** is data that is measured and can have almost any numeric value.

It can include decimal values.

### Examples

- height of kids in third grade classes, such as 52.5 inches or 65.7 inches
- runtime markers in a video
- temperature

### Simple Explanation

Continuous data is measured and can include many possible values.

---

## Discrete Data

**Discrete data** is data that is counted and has a limited number of values.

It usually appears as whole numbers.

### Examples

- number of people who visit a hospital each day, such as 10, 20, or 200
- maximum capacity allowed in a room
- tickets sold in the current month

### Simple Explanation

Discrete data is counted.

---

# Continuous vs Discrete Data

| Data Type | Definition | Examples |
|---|---|---|
| **Continuous Data** | Measured data that can have almost any numeric value | Height, video runtime, temperature |
| **Discrete Data** | Counted data with limited values | Hospital visits, room capacity, tickets sold |

---

# 4. Qualitative Data vs Quantitative Data

## Qualitative Data

**Qualitative data** is descriptive data that explains a quality or characteristic.

It is usually subjective and not measured with numbers.

### Examples

- favorite exercise activity
- brand with best customer service
- fashion preferences of young adults

### Simple Explanation

Qualitative data describes qualities or categories.

---

## Quantitative Data

**Quantitative data** is data that is measured objectively using numbers, quantities, or ranges.

### Examples

- percentage of board-certified doctors who are women
- population size of elephants in Africa
- distance from Earth to Mars at a particular time

### Simple Explanation

Quantitative data measures numbers or amounts.

---

# Qualitative vs Quantitative Data

| Data Type | Definition | Examples |
|---|---|---|
| **Qualitative Data** | Subjective and descriptive measure of a quality or characteristic | Favorite exercise, best customer service brand, fashion preferences |
| **Quantitative Data** | Specific and objective numeric measure | Percentages, population size, distance |

---

# 5. Nominal Data vs Ordinal Data

## Nominal Data

**Nominal data** is a type of qualitative data that is categorized without a set order.

The categories are labels, but they are not ranked.

### Examples

- first-time customer, returning customer, regular customer
- new job applicant, existing applicant, internal applicant
- new listing, reduced price listing, foreclosure

### Simple Explanation

Nominal data names categories without ranking them.

---

## Ordinal Data

**Ordinal data** is a type of qualitative data with a set order or scale.

The categories can be ranked.

### Examples

- movie ratings, such as 1 star, 2 stars, or 3 stars
- ranked-choice voting selections, such as 1st, 2nd, or 3rd
- satisfaction level in a survey, such as satisfied, neutral, or dissatisfied

### Simple Explanation

Ordinal data has order or ranking.

---

# Nominal vs Ordinal Data

| Data Type | Definition | Examples |
|---|---|---|
| **Nominal Data** | Qualitative data categorized without a set order | Customer type, applicant type, listing type |
| **Ordinal Data** | Qualitative data with a set order or scale | Movie ratings, ranked voting, satisfaction levels |

---

# 6. Structured Data vs Unstructured Data

## Structured Data

**Structured data** is data organized in a specific format, usually rows and columns.

It is easy to search, sort, filter, and analyze.

### Examples

- expense reports
- tax returns
- store inventory

### Simple Explanation

Structured data fits neatly into tables.

---

## Unstructured Data

**Unstructured data** is data that cannot easily be stored as rows and columns in a relational database.

It does not follow a simple table format.

### Examples

- social media posts
- emails
- videos

### Simple Explanation

Unstructured data does not fit neatly into rows and columns.

---

# Structured vs Unstructured Data

| Data Type | Definition | Examples |
|---|---|---|
| **Structured Data** | Data organized in a certain format, such as rows and columns | Expense reports, tax returns, store inventory |
| **Unstructured Data** | Data that cannot easily be stored in rows and columns | Social media posts, emails, videos |

---

# Summary of Data Format Classifications

| Classification | Main Difference |
|---|---|
| **Primary vs Secondary** | Whether the data was collected directly by you or by someone else |
| **Internal vs External** | Whether the data comes from inside or outside the organization |
| **Continuous vs Discrete** | Whether the data is measured or counted |
| **Qualitative vs Quantitative** | Whether the data describes qualities or measures numbers |
| **Nominal vs Ordinal** | Whether categories have no order or a ranked order |
| **Structured vs Unstructured** | Whether the data is organized in rows and columns |

---

# Big Lesson

Different data formats serve different purposes.

A Data Analyst needs to recognize the format of the data so they can choose the right data and use the right analysis method.

---

# Key Takeaways

- Data format refers to how data is organized, stored, or presented.
- Primary data is collected directly from first-hand sources.
- Secondary data is collected by someone else or from other research.
- Internal data is stored inside a company’s own systems.
- External data comes from outside the company.
- Continuous data is measured and can have many numeric values.
- Discrete data is counted and has a limited number of values.
- Qualitative data describes qualities, categories, or characteristics.
- Quantitative data measures numbers, quantities, or ranges.
- Nominal data is categorized without a set order.
- Ordinal data is categorized with a set order or scale.
- Structured data is organized in rows and columns.
- Unstructured data does not fit neatly into rows and columns.
- Choosing the right data format helps analysts manage and use data effectively.

---

# Extra Important Notes

- Primary data gives you more control because you collect it yourself.
- Secondary data can save time because it already exists.
- Internal data is often easier to access within a company.
- External data can help compare your organization to outside benchmarks.
- Continuous data is useful for precise measurement.
- Discrete data is useful for counts.
- Nominal and ordinal data are both qualitative, but ordinal data has rank.
- Structured data is usually easier to analyze than unstructured data.
- Unstructured data can still be valuable, but it may require extra processing.

# Final Summary

This reading explains common data format classifications. Data can be primary or secondary depending on who collected it. It can be internal or external depending on whether it comes from inside or outside an organization. Numeric data can be continuous if it is measured or discrete if it is counted. Descriptive data can be qualitative, while numeric data is quantitative. Qualitative data can be nominal if it has no order or ordinal if it has a ranked order. Data can also be structured, meaning it is organized in rows and columns, or unstructured, meaning it does not fit neatly into a table. The main lesson is that understanding data formats helps Data Analysts choose, organize, and analyze data correctly.
