### What are some characteristics of small data? Select all that apply.
- Often organized and analyzed using spreadsheets
- Useful when making day-to-day decisions
- Likely to focus on specific metrics

### Fill in the blank: Big data typically involves large datasets containing data from long _____, which has to be organized before analysis.
- Big data typically involves large datasets containing data from long time periods, which has to be organized before analysis.

### A data professional works with data that is stored in a database. They use SQL to sort and analyze it in order to ensure the insights are as accurate as possible. What type of data are they most likely using?
- They are most likely using big data. Big data is typically stored in a database and requires SQL for sorting and analysis. Spreadsheets are more effective when using small data.

### When considering the benefits and challenges of big data, which of the “four V words” describe the amount of data being analyzed?
- Volume describes the amount of data being analyzed.
