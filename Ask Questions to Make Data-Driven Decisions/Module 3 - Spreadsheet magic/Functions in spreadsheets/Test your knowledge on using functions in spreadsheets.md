# Spreadsheet Functions Assessment Notes

## Question 1: `=SUM(B1:B4)`

**Correct answer:** `15`

### Why
The `SUM` function adds all values in the selected range of cells.

### Key note
- `SUM` = adds numbers in a range
- Example: `=SUM(B1:B4)`

---

## Question 2: Average of cells `F1` to `F10`

**Correct answer:** `=AVERAGE(F1:F10)`

### Why
The `AVERAGE` function returns the average of all values in the selected range.

### Key note
- use a colon `:` to show a range
- `AVERAGE` = finds the mean of the values

---

## Question 3: `=MIN(D1:D4)`

**Correct answer:** `2`

### Why
The `MIN` function returns the smallest numeric value in a selected range.

### Key note
- `MIN` = smallest value
- Example: `=MIN(D1:D4)`

---

## Question 4: Greatest value from `G60` to `G100`

**Correct answer:** `=MAX(G60:G100)`

### Why
The `MAX` function returns the largest numeric value in the selected range.

### Key note
- `MAX` = largest value
- use `:` to include all cells between the two references

---

## Main Functions to Remember

### `SUM`
Adds all numbers in a range.

**Example:**  
`=SUM(B1:B4)`

---

### `AVERAGE`
Returns the average of all numbers in a range.

**Example:**  
`=AVERAGE(F1:F10)`

---

### `MIN`
Returns the smallest number in a range.

**Example:**  
`=MIN(D1:D4)`

---

### `MAX`
Returns the largest number in a range.

**Example:**  
`=MAX(G60:G100)`

---

## Important Spreadsheet Rule

Functions:

- always begin with `=`
- use parentheses `()`
- use a colon `:` to show a continuous range

---

## Short Takeaway

This assessment focused on four basic spreadsheet functions: `SUM`, `AVERAGE`, `MIN`, and `MAX`. These functions are important because they help data analysts quickly calculate totals, averages, smallest values, and largest values from a range of cells.
