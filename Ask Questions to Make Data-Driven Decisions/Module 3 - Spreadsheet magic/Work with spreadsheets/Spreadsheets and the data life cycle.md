# Spreadsheet Basics for Data Analysts

## Overview

Data analysts spend a lot of time:

- organizing data
- sorting and filtering data
- performing calculations
- preparing data for analysis

One of the main tools used for these tasks is a **spreadsheet**.

---

## Why Spreadsheets Are Useful

Spreadsheets help data analysts manage data in an organized way and make calculations more efficiently.

They are useful for:

- storing data
- organizing information
- filtering records
- summarizing data
- calculating results

---

## Example Scenario

Imagine you work for a **construction company**.  
The company asks you to analyze data about its **expenses**.

First, you access the data and add it to a spreadsheet.  
Once the data is in the spreadsheet, you can start preparing it for analysis.

---

## Common Spreadsheet Tasks for Data Analysts

### 1. Organizing Data
A data analyst may begin by organizing the data so it is easier to work with.

One way to do this is by using a **pivot table**.

#### Pivot table
A pivot table is a way to organize and summarize data so it is easier to review and analyze.

### Why it helps
- makes large datasets easier to understand
- groups related information together
- helps summarize data quickly

---

### 2. Sorting and Filtering Data
After organizing the data, the analyst may **sort** or **filter** it.

#### Sorting
Sorting puts data in a particular order, such as:
- highest to lowest
- oldest to newest
- alphabetical order

#### Filtering
Filtering shows only the data needed for a specific analysis.

### Example
If the construction company only wants to review expenses from the **last three months**, the analyst can filter the spreadsheet to show only that time period.

### Why it helps
- removes unnecessary data from view
- helps the analyst focus on the most relevant records
- saves time during analysis

---

### 3. Performing Calculations
Once the right data is visible, the analyst can perform calculations to learn more from it.

### Example
The analyst may want to find out:

- which construction projects cost the most
- total expenses over a time period
- average spending by project

This is where **formulas** and **functions** are useful.

---

## Formulas and Functions

### Formula
A **formula** is an expression used to calculate values in a spreadsheet.

### Function
A **function** is a preset formula that performs a specific calculation.

### Why they are useful
Formulas and functions help analysts:

- do quick math
- save time
- reduce manual work
- analyze data more accurately
