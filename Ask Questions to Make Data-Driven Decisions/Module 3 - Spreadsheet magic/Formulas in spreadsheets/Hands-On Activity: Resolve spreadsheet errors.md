## 2. How does error resolution contribute to data accuracy? Why is this so important to data analysis and the reliability of results?

Error resolution improves data accuracy because it ensures formulas return the correct results instead of misleading values or error messages. This is very important in data analysis because inaccurate calculations can lead to wrong conclusions, poor decisions, and reports that stakeholders cannot trust.

## 3. How do you plan to apply the error identification and resolution skills you've gained in this activity to your future data analysis projects?

In future data analysis projects, I plan to use these skills by checking formulas carefully, reviewing referenced cells, and confirming that the data types and values are correct before drawing conclusions. This will help me catch mistakes early, improve the quality of my analysis, and create more reliable results.
