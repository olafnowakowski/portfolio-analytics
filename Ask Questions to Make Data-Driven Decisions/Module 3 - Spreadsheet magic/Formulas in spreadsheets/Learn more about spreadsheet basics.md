# Spreadsheet Formula Notes
# Key Concepts

## Formula
A formula is an expression used to calculate values in a spreadsheet.

## Cell reference
A cell reference uses the location of a cell, such as `B2` or `D4`, in a formula.

## Relative reference
When you copy a formula, the spreadsheet updates the references based on the new row or column.

## Automatic recalculation
When a referenced cell changes, the spreadsheet updates the formula result automatically.

## Percent formatting
The `%` button changes a decimal result into percentage format.

---

# Quick Formula List

| Task | Formula Example |
|---|---|
| Subtraction | `=2-2` |
| Total sales | `=B2+C2+D2+E2` |
| Average sales | `=(B2+C2+D2+E2)/4` |
| Percent change | `=(E2-D2)/D2` |

---

# Key Takeaways

- All formulas begin with `=`
- Cell references make formulas flexible
- Formulas recalculate automatically when data changes
- Copying formulas saves time
- Averages and percent changes can be calculated with formulas
- Blank cells can cause errors, especially when dividing
- Fixing source data often resolves formula errors

---

# Short Summary

Spreadsheet formulas help analysts calculate totals, averages, and percent changes quickly and accurately. Cell references make formulas dynamic, so results update automatically when data changes. Copying formulas improves efficiency, while checking for blank cells helps prevent calculation errors.
