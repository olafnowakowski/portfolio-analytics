### Kiedy używać wersji MASTER

Wersja **MASTER** nie powinna być Twoim głównym CV do masowej wysyłki.

Używaj jej, gdy:

- oferta jest bardzo ogólna,
- ogłoszenie łączy systemy, procesy i operations,
- wysyłasz CV spontanicznie,
- piszesz do rekrutera bez konkretnej oferty,
- chcesz mieć bazę do szybkiego dopasowania.

### Zasada

> **MASTER = baza robocza / wersja ogólna, a nie domyślne CV do każdej aplikacji.**

## 7. Poziom stanowisk, których warto szukać

| Poziom | Szukać? | Komentarz |
|---|---|---|
| **Junior / Młodszy / Junior** | Tak | Dobre, jeśli oferta jest rozwojowa albo mocniej techniczna |
| **Specialist / Specjalista / Spezialist** | Tak, najbardziej | To powinien być główny poziom |
| **Coordinator / Koordynator / Koordinator** | Tak | Bardzo dobry fit pod operations i support |
| **Regular / Mid-level / Associate** | Tak | Jeśli zakres obowiązków jest sensowny |
| **Senior / Starszy / Senior** | Ostrożnie | Tylko jeśli wymagania nie są przesadnie wysokie |
| **Manager / Lead / Head** | Raczej nie | Chyba że ogłoszenie jest miękkie i mało managerskie |
